### Added keyboard support
 - 0-9 (incl NumpadKeys): to corresponding digits
 - Decimal point '.' - . key
 - '+' operator : + key
 -  '-' operator : - key
 - Negative sign: - key (can only be placed before numbers or the operators "x" or "/")
 - "x" operator: * or x key
 - "/" operator: / key
 - Clear: c
 - Delete:  Backspace
 - Equals '=':  Enter.
